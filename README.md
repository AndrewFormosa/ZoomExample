## ZOOM Project _ Java programing

### Personal Project Aims

To create a 3D vector graphics environment from simple geometrical mathmatics.

https://github.com/AndrewFormosa/ZoomExample/blob/master/ZoomExample.jar

## View & Controls

Open the JAR file.
Use keys:
B-Speed up
N - Stop
Q - Point UP
A - Point Down
Y - Bank Left
U - Bank Right
T - Fire.
